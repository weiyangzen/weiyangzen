No-comment round zero must not trigger manual PR comments; comments-present round zero must commit, push, and tag bots. Both task templates prohibit modifying `.humanize/pr-loop/` state files (`prompt-template/pr-loop/round-0-task-has-comments.md:30-35`, `prompt-template/pr-loop/round-0-task-no-comments.md:18-23`).
