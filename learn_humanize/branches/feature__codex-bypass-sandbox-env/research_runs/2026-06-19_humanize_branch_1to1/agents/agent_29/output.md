It does not mutate state; it transitions attempted reads into allowed, hard-error, or blocked outcomes.

Round extraction only recognizes numeric `round-N-{summary,prompt,todos}.md`. Path checks are string-based, so symlink/canonical path equivalence is not resolved. `finalize-state.md` overrides `state.md` during finalize. The allowlist is exact active-loop path matching only.
