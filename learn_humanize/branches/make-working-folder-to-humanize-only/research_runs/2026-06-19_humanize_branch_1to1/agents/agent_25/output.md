The hook extracts `FILE_PATH`, lowercases it for pattern checks, finds `ACTIVE_LOOP_DIR`, reads `CURRENT_ROUND` from `state.md`, extracts `CLAUDE_ROUND` from the filename, and derives `FILE_TYPE` from the suffix ([hooks/loop-read-validator.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/make-working-folder-to-humanize-only/hooks/loop-read-validator.sh:22)). The round-number invariant is enforced at `CLAUDE_ROUND != CURRENT_ROUND`; if violated, the hook renders `block/wrong-round-file.md` through `load_and_render_safe` and exits `2` ([hooks/loop-read-validator.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/make-working-folder-to-humanize-only/hooks/loop-read-validator.sh:103)).
