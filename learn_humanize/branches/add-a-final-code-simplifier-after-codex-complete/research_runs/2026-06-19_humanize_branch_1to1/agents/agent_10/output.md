Its output becomes state-machine control text: block reasons for validators, Codex/Claude prompts, and finalize-phase instructions. `hooks/lib/loop-common.sh` sources this library, initializes `TEMPLATE_DIR`, and warns but does not fail when validation fails (`hooks/lib/loop-common.sh:46`, `:51`, `:54`). `hooks/loop-codex-stop-hook.sh` uses it to produce the finalize block reason and next-round prompt files (`hooks/loop-codex-stop-hook.sh:1005`, `:1075`).
