- core_role: `scripts/rlcr-stop-gate.sh` is a non-hook CLI wrapper around the authoritative RLCR Stop hook, `hooks/loop-codex-stop-hook.sh`. Its purpose is to let skill workflows or other non-native hook callers run the same stop-gate enforcement and phase-transition logic as a real Claude/Codex Stop event, while exposing simple wrapper exit codes: allow `0`, block `10`, wrapper/runtime error `20`. The role is explicitly documented at [scripts/rlcr-stop-gate.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/use-realpath4everything/scripts/rlcr-stop-gate.sh:3) and [scripts/rlcr-stop-gate.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/use-realpath4everything/scripts/rlcr-stop-gate.sh:8).

The default resolver is deterministic: `CLAUDE_PROJECT_DIR`, then `git rev-parse --show-toplevel`, with no `pwd` fallback; it canonicalizes through `realpath`, which is central to this branch’s path-realization behavior.
