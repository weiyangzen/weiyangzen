The detector receives already-lowercased Bash command input, splits chained commands on shell operators such as `&&`, `||`, pipe, and semicolon, then scans each segment for `git ... add` (`hooks/lib/loop-common.sh:1262-1305`). It extracts arguments after `add`, strips quotes for path matching, and blocks direct `.humanize` path references without over-blocking names such as `.humanizeconfig` (`hooks/lib/loop-common.sh:1308-1323`). It separately detects force flags, all flags, and broad-scope targets (`hooks/lib/loop-common.sh:1325-1346`). Non-force broad-scope `git add .` or `git add *` is blocked when `.humanize` exists and is not ignored (`hooks/lib/loop-common.sh:1366-1372`).
