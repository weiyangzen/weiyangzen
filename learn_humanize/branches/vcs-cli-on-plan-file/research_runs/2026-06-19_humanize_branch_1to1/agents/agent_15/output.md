- core_role: Declarative Claude Code hook registry for the Humanize RLCR loop. The file does not implement shell logic itself; it is the dispatch table that makes the RLCR state machine enforceable by attaching validators and the Codex review stop hook to Claude lifecycle events. - inputs_outputs_state: Inputs are Claude hook events and hook JSON payloads supplied by Claude Code, plus the `CLAUDE_PLUGIN_ROOT` environment variable used to resolve every command path. The registered scripts then consume RLCR files under `.humanize-loop.local/<timestamp>/`, especially `state.md`, `plan-backup.md`, `goal-tracker.md`, round prompts, summaries, and review results. Setup creates those state files with `current_round`, `max_iterations`, Codex config, `push_every_round`, `commit_plan_file`, `plan_file`, `plan_file_tracked`, and `start_commit` ([scripts/setup-rlcr-loop.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/scripts/setup-rlcr-loop.sh:326)). State transitions are performed by referenced scripts: `stop_loop` renames active `state.md` to `<prefix>-state.md` with prefixes `completed`, `stopped`, `unexpected`, or `cancelled` ([hooks/lib/loop-common.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/hooks/lib/loop-common.sh:23)); the stop hook increments `current_round` when Codex finds issues ([hooks/loop-codex-stop-hook.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/hooks/loop-codex-stop-hook.sh:865)), marks completion by stopping with `completed` on final-line `COMPLETE` ([hooks/loop-codex-stop-hook.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/hooks/loop-codex-stop-hook.sh:820)), and stops with `stopped` on `STOP` or max iterations ([hooks/loop-codex-stop-hook.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/hooks/loop-codex-stop-hook.sh:512), [hooks/loop-codex-stop-hook.sh](/Users/wangweiyang/GitHub/humanize_branch_worktrees/vcs-cli-on-plan-file/hooks/loop-codex-stop-hook.sh:831)).
