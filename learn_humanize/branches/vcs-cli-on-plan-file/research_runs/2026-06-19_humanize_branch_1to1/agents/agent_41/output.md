- algorithmic_behavior: The template renders a fixed “Incomplete Todos Detected” block with one variable placeholder, `{{INCOMPLETE_LIST}}`, inserted between the explanatory sentence and the required action list. The Stop hook invokes `check-todos-from-transcript.py`; if that checker exits `1`, the hook extracts the checker output after the first marker line, renders this template, and returns JSON with `"decision": "block"` instead of continuing to Codex review.
