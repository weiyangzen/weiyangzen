- gates_or_invariants: Requires PR number for `codex-thumbsup`, comment id for `claude-eyes`, rejects unknown options and duplicate positional args (`scripts/check-bot-reactions.sh:92-117`, `197-230`). Claude reaction API errors are retried rather than fatal during attempts (`scripts/check-bot-reactions.sh:273-278`). Older Codex reactions deliberately return `1` when `--after` is set.
- validation_or_tests: Covered indirectly by `tests/test-pr-loop-stophook.sh` via PR stop-hook scenarios: Codex thumbs-up approval (`tests/test-pr-loop-stophook.sh:896-1009`), Claude eyes timeout (`1011-1143`), stale trigger/force-push timestamp behavior (`21-138`, `538-640`), and fork PR lookup (`1390-1517`).
