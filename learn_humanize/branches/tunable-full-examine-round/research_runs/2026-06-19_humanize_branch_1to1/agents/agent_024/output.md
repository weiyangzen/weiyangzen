- algorithmic_behavior: On `UserPromptSubmit`, the manifest runs `loop-plan-file-validator.sh` (`hooks/hooks.json:4-10`). These validators gate tool access before the agent can read/write/edit/execute shell commands that could mutate loop state, prompt files, summaries, goal trackers, plan backups, or PR-loop protected files. On `Stop`, it runs both RLCR and PR-loop stop hooks, each with a 7200-second timeout (`hooks/hooks.json:52-64`).
