- core_role: This file is a Codex review prompt template for mandatory Full Alignment Check rounds in the RLCR loop. It is algorithmically relevant because it defines the review contract that controls whether iterative development continues, completes, or trips a stagnation circuit breaker. The template is selected instead of the regular review prompt when the current round satisfies the configured full-review cadence in `hooks/loop-codex-stop-hook.sh:773-783`, then rendered to `.humanize/rlcr/<timestamp>/round-N-review-prompt.md` via `load_and_render_safe` at `hooks/loop-codex-stop-hook.sh:816-829`.
- algorithmic_behavior: The prompt forces a multi-part audit. First, it requires Codex to read the original implementation plan before review (`prompt-template/codex/full-alignment-review.md:5-10`). It embeds Claude's round summary through `{{SUMMARY_CONTENT}}` (`prompt-template/codex/full-alignment-review.md:13-16`). It then requires a goal-tracker audit across acceptance criteria, forgotten items, deferred items, and completion summary (`prompt-template/codex/full-alignment-review.md:19-48`). Next it requires a deep implementation review comparing Claude's claims to reality and consulting design docs (`prompt-template/codex/full-alignment-review.md:50-55`). Finally, it adds a historical progress audit for full alignment rounds and asks Codex to decide whether repeated review/development loops have stagnated (`prompt-template/codex/full-alignment-review.md:59-84`).
