Blocks `git push` unless `push_every_round` is true (`hooks/loop-bash-validator.sh:96-105`). Blocks `git add` patterns that would stage `.humanize` (`hooks/loop-bash-validator.sh:109-118`). Blocks PR-loop `state.md` and generated read-only PR round files by full path or bare basename (`hooks/loop-bash-validator.sh:403-442`).
