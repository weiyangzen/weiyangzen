Examples: `hooks/loop-read-validator.sh:127-167` renders wrong file/location/round/directory blocks; `hooks/loop-write-validator.sh:152-263` renders protected state, plan backup, goal tracker, wrong summary, wrong round, and wrong directory blocks; `hooks/loop-edit-validator.sh:117-178` renders protected state, plan backup, goal tracker, and wrong round blocks; `hooks/lib/loop-common.sh:347-405` centralizes todos/prompt/state/finalize/summary/goal-tracker block messages; `hooks/lib/loop-common.sh:793-809` covers PR-loop protected files; `hooks/lib/loop-common.sh:932-969` covers `.humanize` git-add protection; `hooks/loop-codex-stop-hook.sh:247-621` renders plan-modified, incomplete-todos, git-status-failed, large-files, git-not-clean, unpushed-commits, work-summary-missing, and goal-tracker-not-initialized blocks.
