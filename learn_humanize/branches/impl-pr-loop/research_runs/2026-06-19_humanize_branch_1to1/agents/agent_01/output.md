- gates_or_invariants: Only PR-loop state under `.humanize/pr-loop` is targeted, not RLCR (`scripts/cancel-pr-loop.sh:55`). Cancellation requires the newest loop directory to contain `state.md`; directories with only terminal state files are not active. Missing metadata is tolerated and displayed as `?` (`scripts/cancel-pr-loop.sh:88`, `scripts/cancel-pr-loop.sh:104`, `scripts/cancel-pr-loop.sh:110`).
