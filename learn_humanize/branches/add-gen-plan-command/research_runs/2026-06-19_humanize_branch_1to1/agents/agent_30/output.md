GNU-compatible paths simply exec `gtimeout "$timeout_secs" "${cmd[@]}"` or `timeout "$timeout_secs" "${cmd[@]}"` (`scripts/portable-timeout.sh:38-45`). Python fallback runs `subprocess.run(sys.argv[1:], timeout=...)` and returns the child exit code or `124` on timeout to match GNU timeout semantics (`scripts/portable-timeout.sh:47-61`). With no implementation, it emits a stderr warning and runs the command unbounded (`scripts/portable-timeout.sh:64-68`).
