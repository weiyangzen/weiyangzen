- inputs_outputs_state: Inputs are template variables `ACTION`, `FILE_PATH`, and `CORRECT_PATH`. - gates_or_invariants: The template enforces the invariant that round files must be accessed from the active loop directory, not older loop directories or same-named files elsewhere. `hooks/loop-read-validator.sh:131-145` computes `CORRECT_PATH="$ACTIVE_LOOP_DIR/$CLAUDE_FILENAME"` and renders this template when `FILE_PATH != CORRECT_PATH`. `hooks/loop-write-validator.sh:201-216` applies the same directory gate for writes after summary round validation.
